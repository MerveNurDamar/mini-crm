# Kullanici Kilavuzu

## Musteri Yonetimi
- Musteri ekle, listele, guncelle, sil.
- Zorunlu alanlar: ad + (telefon veya email).
- Soyad ve adres opsiyoneldir.

## Urun/Stok Yonetimi
- Urun ekle, guncelle, listele.
- Stok takibi acik olan urunlerde stok dusumu uygulanir.

## Siparis Yonetimi
- Siparis olusturma ve durum guncelleme.
- Stok yetersizse siparis beklemeye alinabilir.

## ETL (Musteri Import)
- CSV/Excel dosyasi import edilir.
- Hatalar raporlanir, duplicate kayitlar atlanir.
- Raporlar `reports/` altina yazilir.
